About this Base

Introduction

Hello. My name is Tyler Wayne, and I will be your host for this website. I'm the coder responsible for the products featured here on Code Haven. I do all of this programming in my spare time, because it's what I love to do. Programming is problem solving, so when I see a problem that can be solved with a program, I do it.

My Work

Every program I've ever created has always been free. I don't think I need to be paid for doing something that I love so much. Another reason for freedom is the fact that programs should be helpful, and emptying your wallet isn't very much of a help.

Most of my programs are also open source, which means that anyone can use them, study them, or even duplicate and change them. It's free as in "free speech," not just "free food." If you want to check out the source code, just head over to GitHub. You can work alone or suggest changes to any of my programs; keep it to yourself or share it with the world. Your call.

Code Haven (the website itself)

I'm sure you can tell by yourself, but Code Haven is a wee bit dull right now. There's no stylesheet powering these pages; it's nothing more than XHTML 1.0. I'll be gradually improving it, so there is some good news.

As far as the content of the site goes, there really isn't any right now. Pretty soon though, there will be information and screenshots of my Chrome extensions and programs, as well as interesting things that I find share-worthy.
